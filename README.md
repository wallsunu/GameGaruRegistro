# GameGaru Registro

Proyecto web desarrollado en **ASP.NET Core MVC**.

Este sistema permite registrar jugadores en la plataforma **GameGaru** y verificar si realizaron el **pago de matrícula** antes de completar el registro.

## Tecnologías utilizadas
- ASP.NET Core
- C#
- Visual Studio Code
- Git
- GitHub

## Funcionalidad
- El formulario permite registrar jugadores.
- Válida si el pago de matrícula fue realizado o no.
- Muestra mensajes según el estado del pago:
  - **Ha completado su registro**
  - **Usted aún no ha completado su pago**
- Permite agregar información como:
  - Nombre del jugador
  - Fecha de nacimiento
  - Hobbies
  - Monto de matrícula pagado
